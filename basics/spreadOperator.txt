‚úÖ What is the Spread Operator? (...)
üëâ Spread (...) expands an array, object, or string into individual elements.
Think of it like ‚Äúopening the box‚Äù.
Example: console.log(...[1, 2, 3]);
Output: 1 2 3

The array [1,2,3] is opened into separate values.

üéØ WHERE Spread Operator is used?

There are 3 major uses:
1Ô∏è‚É£ Copy arrays (shallow copy)
const arr1 = [1, 2, 3];
const arr2 = [...arr1];
console.log(arr2); // [1, 2, 3]

Without spread:
const arr2 = arr1; // ‚ùå reference copy
Both point to the same array.

2Ô∏è‚É£ Merge arrays
const a = [1, 2];
const b = [3, 4];
const c = [...a, ...b];
console.log(c); // [1, 2, 3, 4]

3Ô∏è‚É£ Add items to array easily
const arr = [1, 2, 3];
const updated = [0, ...arr, 4];
console.log(updated);
// [0, 1, 2, 3, 4]

4Ô∏è‚É£ Copy objects (shallow copy)
const obj1 = { name: "Abhinav", age: 25 };
const obj2 = { ...obj1 };
console.log(obj2);
// { name: "Abhinav", age: 25 }

5Ô∏è‚É£ Merge objects
const o1 = { a: 1 };
const o2 = { b: 2 };
const o3 = { ...o1, ...o2 };
console.log(o3); // { a: 1, b: 2 }

6Ô∏è‚É£ Pass array values into a function

Old way:
Math.max.apply(null, [1, 2, 3]);

Modern way:
Math.max(...[1, 2, 3]); // 3
Much cleaner.

7Ô∏è‚É£ Convert string into array (spread chars)
console.log([..."ABHI"]);
// ["A", "B", "H", "I"]

üß† Important Note
Spread operator does shallow copy, not deep copy.

Meaning:

const a = [{x: 1}, {y: 2}];
const b = [...a];

a[0].x = 99;
console.log(b[0].x);  // 99 ‚ùó

Because inner objects are references, not copied.
